📌 Student Result Management System
f. System Scope

The Student Result Management System is used to store, manage, and display student academic results. It is mainly used by teachers, students, and administrators. Teachers can upload marks, students can view results, and admins manage the system. This system helps reduce manual work and ensures accuracy.

g. Functional Requirements (exactly 3)
The system must allow teachers to input and update student marks.
The system must allow students to view their results online.
The system must allow admin to manage users and subjects.
h. Non-Functional Requirements (exactly 2)
The system should be secure to protect student data.
The system should be fast and responsive.
i. Priority Analysis
Requirement	Priority	Justification
Input/Update Marks	High	Core functionality without this system is useless
View Results	High	Students must access results easily
Admin Management	Medium	Important but not as critical as marks/results
